# 🎀 Maimero Timer

まいめろといっしょにがんばろう♪

かわいいパステルデザインのカウントダウンタイマー & ストップウォッチアプリです。

## ✨ 機能

- ⏳ **カウントダウンタイマー** — 5/10/25/45/60分のプリセット or 手動設定
- ⏱ **ストップウォッチモード** — 切り替え可能
- 🎂 **セッション統計** — 完了回数・合計時間・連続達成を記録
- 🎊 **完了アニメーション** — コンフェッティ＆メロディー
- 📳 **バイブレーション** — 残り5秒・完了時
- 💾 **統計の永続化** — localStorage で保存

## 🎨 デザイン

- パステルピンク × ブルー × パープルのフラットデザイン
- モバイルファースト（最大幅 420px）
- フローティングアニメーション背景
- グラデーション SVG プログレスリング
